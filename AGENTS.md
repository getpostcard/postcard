# Agent instructions

Instructions for AI coding agents (Claude Code and others) working in this repository.
CLAUDE.md is a symlink to this file, so agents that look for either name find the same
instructions.

## Commit attribution

Any commit authored or assisted by an AI agent must include attribution in its message:
a `Co-authored-by:` trailer naming the agent, on its own line in the trailer block. For
example:

    Co-authored-by: Claude <noreply@anthropic.com>

Use the attribution your agent or model documents. Some include the model name, such as
`Co-authored-by: Claude Opus 4.8 <noreply@anthropic.com>`. Commits written entirely by a
human do not need the trailer.
